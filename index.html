Projects:

  • Tutor Scheduling App

    Used in Production: March 2025 - Present

    This is the most complete project on this page. It's an ASP.Net web app with a postgres database I wrote for my tutoring center, so we can easily answer the question "Who can help with this subject?" both future appointments, and to pick tutors to help with less common courses. I made it March 2025, and originally hosted it on Azure, before moving it to Heroku to lower costs. We've had this app open in a tab at the front desk of the tutoring center since it was made.

    App features:
    • Dashboard of on duty tutors. (or show the dashboard for a specific time/day.)s
    • Upload schedule as a csv file.
    • Can add/remove tutors and courses.
    • Utility to see how often one tutor's schedule overlaps with another's (used to determine which supervisor should associate with which tutor. i.e. for evaluations.)

  • "Bulwark" (Partial MMO Style Game)

    **NOTE: This code will not compile. It's only snippets taken from a project.**

    This some of the code from one a (singleplayer) MMO style game I started. I didn't use GitHub for Unity projects, using Unity VCS instead, so the code on GitHub is incomplete. Also, because I can't make store assets public GitHub, I extracted just my code here.

    It contains code for a Scriptable Object Architecture to support rebindable keybinds, a MMO style movement for grounded, air, and sliding movement, and a MMO style tab target system.

    It uses a dependency injection framework called "Init(args)", and there are both edit and play mode tests for movement, targeting, the input system, and for other various helpers.

  • Basic Sudoku Solver with Blazor

    Sudoku was popular in the tutoring center around when I tried Blazor, so I made a very simple, quick, and dirty Sudoku solver. It mostly does naive solving (i.e. fill squares that only one option with that option.) It connects to an external API to load puzzles, or you can enter your own puzzle manually.

Project:

  • Gleam Math Engine

    A practice project to try out gleam. I thought it would be neat to have a tool that could simplify expressions and solve equations, ideally by following steps like humans do (i.e. PEMDAS), rather than just outputting a black box answer like a calculator. It handles some the P, M, and A in PEMDAS, and I learned a lot about basic parsers.
